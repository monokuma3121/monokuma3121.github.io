# vite 创建 vue3 项目

先看一个创建 vue3 项目的命令： vue create 项目名称 ，这是基于 webpack 构建工具创建的，它有一个问题，就是 webpack 创建的项目编译慢
所以，我们使用 vite 构建工具：

  1. 创建项目
# pnpm
pnpm create vite

# yarn
yarn create vite

# npm
npm init vite@latest
  1. 启动项目
cd 项目所在位置 && yarn && yarn dev
  1. 为保证 node 的使用
  1. 修改 tsconfig.json
  1. 修改 vite.config.ts

# vue2 和 vue3 的区别

  1. v-if 与 v-for 优先级
  • 2.x 中， v-for > v-if
  • 3.x 中， v-if > v-for
    由于语法歧义，建议避免在同一元素上使用两者。
  1. v-for 中的 Ref 数组
  • 2.x 中，会自动把 ref 填充内容
  • 3.x 中，需要手动添加
