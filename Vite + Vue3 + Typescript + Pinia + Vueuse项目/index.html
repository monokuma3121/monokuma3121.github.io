# 3. 集成 pinia

Pinia 读音：['piːnə]，是 Vue 官方团队推荐代替 Vuex 的一款轻量级状态管理库。

Pinia 有如下特点：

  • 完整的 typescript 的支持；
  • 足够轻量，压缩后的体积只有 1.6kb;
  • 去除 mutations，只有 state，getters，actions（这是我最喜欢的一个特点）；
  • actions 支持同步和异步；
  • 没有模块嵌套，只有 store 的概念，store 之间可以自由使用，更好的代码分割；
  • 无需手动添加 store，store 一旦创建便会自动添加；

# 安装

 pnpm i pinia --save

# 使用
